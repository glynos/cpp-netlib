C++ Network Library¶

Authors:Glyn Matthews <glyn.matthews@gmail.com> Dean Michael Berris <mikhailberis@gmail.com>
Date:Aug 27, 2010
Version:0.7
Description:Complete user documentation, with examples, for the cpp-netlib.
Copyright:Copyright Glyn Matthews, Dean Michael Berris 2008-2010. Distributed under the Boost Software License, Version 1.0. (See accompanying file LICENSE_1_0.txt or copy at http://www.boost.org/LICENSE_1_0.txt)

The cpp-netlib is a library that is intended to provide application layer protocol support (HTTP, XMPP etc.) using modern C++ techniques.
